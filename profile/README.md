## 📄 Overview

<a href="#iCADMac">iCADMac</a> is a professional CAD application for Mac that provides native AutoCAD compatibility — running AutoCAD commands, opening and saving DWG files without conversion, and providing a familiar CAD environment for engineers, architects, and designers who need full AutoCAD compatibility on Mac without running Windows virtualization.

The <a href="#iCADMac">native DWG compatibility</a> is iCADMac's defining technical requirement — reading and writing DWG files in the same format that AutoCAD produces, without format conversion, file translation, or compatibility loss. DWG files from clients, consultants, and collaborators open directly, edit natively, and save back to DWG without the round-trip conversion issues that can occur with translating applications. This native compatibility means iCADMac-produced files are indistinguishable from AutoCAD files for downstream recipients. The <a href="#iCADMac">AutoCAD command set compatibility</a> accepts the same keyboard commands that AutoCAD users have memorized — LINE, CIRCLE, TRIM, EXTEND, OFFSET, HATCH, BLOCK, XREF, and the hundreds of other AutoCAD commands execute in iCADMac as they do in AutoCAD. Users who know AutoCAD find iCADMac immediately productive without relearning a new command vocabulary.

The <a href="#iCADMac">familiar ribbon interface</a> replicates the AutoCAD ribbon layout — panels, tools, and command access organized in the same structure that AutoCAD users navigate, reducing the interface learning required when switching from AutoCAD on Windows to iCADMac on Mac. The <a href="#iCADMac">LISP and DCL scripting support</a> runs AutoLISP and DCL scripts written for AutoCAD — the custom macros, automated drafting routines, and specialized tools that many engineering and architectural practices have developed over years of AutoCAD use execute in iCADMac, preserving the productivity value of existing scripting investments.</p>

<a href="#iCADMac">2D drafting completeness</a> covers the full range of 2D CAD operations — drawing entities, editing commands, dimensioning, annotation, layers, blocks, external references, hatching, and every other 2D CAD capability that production drafting requires. <a href="#iCADMac">3D modeling capabilities</a> extend to solid modeling, surface modeling, and 3D viewing for the three-dimensional design work that architecture and mechanical engineering increasingly require alongside traditional 2D documentation.

<a href="#iCADMac">PDF underlay support</a> attaches PDF files as underlays in drawings — tracing over PDF drawings, using PDF site plans as references, and working with PDF-delivered drawings as tracing bases without manual redrawing. <a href="#iCADMac">Rendering and visualization</a> produces photorealistic renders from 3D models for presentation and visualization — generating client presentation images from the 3D model without exporting to a separate rendering application.</p>
